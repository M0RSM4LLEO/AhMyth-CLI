## <div align="center">Usage</div>
1. Clone it:
```
   git clone https://github.com/M0RSM4LLEO/AhMyth-CLI
```
&nbsp;

2. Open it:
```
cd AhMyth-CLI
```
&nbsp;

3. Run it:
```
./ahmyth-cli
```
